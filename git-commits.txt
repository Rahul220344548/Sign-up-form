feat → new feature
fix → bug fix
style → CSS / formatting (no logic change)
refactor → code cleanup
docs → documentation
chore → setup / assets / config

E.g. 
git commit -m "chore: add images to img folder and custom fonts"

feat: add basic signup form structure
style: add CSS layout for form
feat: implement form validation
chore: add images and fonts assets
feat: create semantic HTML structure for signup form
feat: add signup form fields and layout structure

refactor: rename CSS classes for clarity
refactor: update HTML class names to follow BEM
refactor: improve naming consistency in layout
